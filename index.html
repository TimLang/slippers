slippers by Sarah Taraporewalla

A strict templating language for ruby, based on string template's philosophy and using it's syntax. See the slippers vs string template for more information on these two templating libraries.

Install

Slippers has been released on gemcutter. To install, make sure you have gemcutter in your .gemrc file and then run 

$ sudo gem install slippers
